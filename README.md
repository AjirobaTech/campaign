# AJÍRÓBA - Official Campaign 2026

This is the official landing page for the AJÍRÓBA Technologies campaign.

## Features

- **Spin & Win**: Interactive giveaway experience.
- **Entry Form**: User registration for participation.
- **CEO Message**: Message from Yeye Ajiroba.
- **About Section**: Information about the campaign security and rewards.

## Technologies Used

- **Vite**: Build tool for fast development.
- **React**: UI library.
- **TypeScript**: Static typing.
- **Tailwind CSS**: Utility-first styling.
- **shadcn/ui**: Accessible UI components.
- **Framer Motion**: Animations.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm i`.
3. Start the dev server: `npm run dev`.
4. Open the browser at `http://localhost:8080`.
